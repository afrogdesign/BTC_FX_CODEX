# Code of Conduct

## Our pledge

We aim to keep this project practical, respectful, and safe for contributors.

BTC FX CODEX is related to financial-market data, AI-generated summaries, and operational monitoring. Discussions should stay focused on improving software quality, safety, reproducibility, and documentation.

## Expected behavior

- Be respectful and specific
- Criticize code and ideas, not people
- Share reproducible information when possible
- Redact private logs, keys, account information, and credentials
- Avoid presenting market opinions as guaranteed facts
- Clearly separate software issues from trading advice

## Unacceptable behavior

- Harassment, insults, or personal attacks
- Posting secrets, private logs, or credentials
- Encouraging unsafe financial decisions
- Misrepresenting generated analysis as guaranteed trading signals
- Spam, scams, or promotional content unrelated to the project

## Enforcement

The maintainer may edit, hide, close, or remove issues and comments that create security, privacy, legal, or safety risks.

The maintainer may also limit participation from users who repeatedly ignore these rules.

## Scope

This code of conduct applies to GitHub issues, pull requests, discussions, documentation, and other project-related communication.
