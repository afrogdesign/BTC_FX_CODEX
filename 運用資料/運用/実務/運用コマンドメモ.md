# BTC Monitor 運用コマンドメモ

更新日: 2026-04-10 JST

いま本当に使うコマンドだけを残したメモです。

前提:
- 日常の主観測先は `iMac 2019`
- `MBP2020` の `Ver02.1` は 2026-03-31 に停止・凍結済み
- `MBA M4` は必要時の手動実行に寄せる
- 現行版は `Ver02.4-v1`

## 再発防止メモ

- `Ver02.4-v1` の常設先は `iMac 2019`
- `MBA M4` は同じ `CODEX` パスを持つが、`Ver02.3-OBS` の常設確認先としては扱わない
- BTC Monitor の手動実行と backtest 再検証は、原則 `cd /Users/marupro/CODEX/BTC_FX_CODEX/btc_monitor && .venv312/bin/python ...` を使う
- システムの `python3` は使わない。今回も `python3` では `pandas` 不足で失敗した
- 比較 CSV の再生成は MEXC からローソク足を取り直すため、ネット到達できない環境では失敗する
- 定刻実行確認は、まず `launchctl print gui/$(id -u)/com.afrog.btc-monitor`、次に `logs/heartbeat.txt`、`logs/last_result.json` の更新時刻を見る
- AI 事後評価画像は `logs/review/chart_snapshots/` に保存される
