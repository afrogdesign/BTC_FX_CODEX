# 運用資料/運用 入口

このフォルダは、BTC 監視の運用実務だけを置く入口です。  
現行の主対象は `Ver02.4-v1` です。履歴レポートや旧版の経緯説明はここではなく、上位の `運用資料/README.md` からたどります。

## まず開くもの

- ルール確認
  - `ルール/今後の運用ルール.md`
  - `ルール/記録ファイル運用ルール.md`
- 日常コマンド
  - `実務/運用コマンドメモ.md`
- AI 事後評価の運用
  - `../計画/AI事後評価運用_Ver02.4-v1.md`
- `Phase 1` 条件の読み方
  - `../計画/Phase1条件の見方.md`
- 閾値確認
  - `調整/採点調整シート.md`

## ここで見る内容

- 毎日どのコマンドを回すか
- どの資料を正本として見るか
- `SL` `TP` `15分足` の価格精度をどう見直すか
- AI 事後評価をどう溜めるか

## このフォルダに残すもの

- BTC 固有の運用ルール
- 日常で本当に使うコマンド
- 調整時にすぐ見る最小メモ

## ここに残さないもの

- Mac 横断の実行ファイル一覧
- 本番/開発機の正本整理
- 古い版の経緯説明
- 日次レポート本文そのもの

これらは `Global_BOX` または上位の `運用資料` を正本にします。
