# 運用資料/運用 フォルダ案内

このフォルダは、運用関連の実務資料を役割別に分けて置く入口です。

## フォルダ構成

- `ルール/`
  - 運用判断や記録運用の正本
- `環境/`
  - 本番常駐状況、開発環境一覧などの環境確認資料
- `実務/`
  - 日常で使うコマンド、ログ確認、改善運用の手順
- `調整/`
  - 判定しきい値や採点見直しの調整資料

## まず開くもの

- 運用ルールを確認したいとき
  - `ルール/今後の運用ルール.md`
  - `ルール/記録ファイル運用ルール.md`
- 環境の現在地を確認したいとき
  - `環境/本番常駐メモ.md`
  - `環境/開発環境_実行ファイル一覧.md`
- 日常運用の手順を見たいとき
  - `実務/運用コマンドメモ.md`
  - `実務/ログ検証と改善運用ガイド.md`
- 判定を調整したいとき
  - `調整/採点調整シート.md`
