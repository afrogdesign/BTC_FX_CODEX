## archive の使い方

- `archive/implemented/`
  - 実装済み、または運用へ移った設計書。
- `archive/superseded/`
  - 現行サマリやロードマップに置き換えられた古い計画。
- `履歴/`
  - 旧版差分、旧設計、過去の構想。
