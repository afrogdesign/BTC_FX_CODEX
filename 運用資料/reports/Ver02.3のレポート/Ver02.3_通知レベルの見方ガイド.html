通知レベルの見方ガイド

いま人間が見るべき主ラベルは、最終ランクの5段階だけです。旧来の 執行可 / 監視 / 無効 / 注意報 / 中立 は残っていますが、これはもうランクではなく、最終ランクの中身を補足する説明用ラベルです。

最終ランク

主表示 件名、本文、詳細HTMLの先頭で使う、人間向けの最終分類です。

  • 🔥 強い本通知
  • 🟠 高め本通知
  • 📊 通常の本通知
  • 👀 注意報
  • ⚪ 送信なし
補足状態

補足説明 「なぜそのランクか」を補足する意味ラベルです。

  • 執行可 = 条件成立
  • 監視 = 条件接近
  • 無効 = 執行不可
  • 注意報 = 方向変化の早期共有
  • 中立 = 方向優位なし
メール送信

送る / 送らない の判定は最終ランクで読みます。

  • 🔥 / 🟠 / 📊 / 👀 はメールあり
  • ⚪ はメールなし
  • 補足状態だけでは送信有無は決まらない

一番大事な見方

見出し いまの意味 メールが来るか 人間の読み方
🔥 強い本通知 条件がかなり整った main 来る かなり有力な本通知として扱う
🟠 高め本通知 一段強い main 来る 本通知だが最上位ではない
📊 通常の本通知 標準的な main 来る 通常の本通知として扱う
👀 注意報 early warning の attention 来る 売買推奨ではなく早め共有として扱う
⚪ 送信なし メール条件未成立 来ない ここはログだけ残る
重要: 監視 はもうランクではありません。たとえば「🟠 高め本通知 + 監視」は普通にありえます。読み方は「ランクは高め本通知、その中身は監視段階」です。

具体例

ケース 最終ランク 補足状態 読み方
条件がかなり整った本通知 🔥 強い本通知 執行可 かなり有力で、条件も成立している
条件が寄ってきた本通知 🟠 高め本通知 監視 本通知だが、まだ監視段階
方向はあるが今は入れない 📊 通常の本通知 無効 本通知は来るが、現時点では執行不可
早めの注意だけ共有 👀 注意報 注意報 本通知ではなく、早めの注意共有
材料不足で送らない ⚪ 送信なし 中立など メール自体が来ない

短く言うと

  • 最初に見るのは 最終ランク
  • 次に見るのは 補足状態
  • 補足状態は「主ラベル」ではなく、ランクの理由説明

式で書くと: 最終ランク = 人間向けの出口、補足状態 = その出口の中身の説明 です。
