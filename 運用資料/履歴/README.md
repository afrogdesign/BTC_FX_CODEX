# 履歴フォルダ

このフォルダは、日常の入口から外した履歴資料を置く場所です。

## ここに置くもの

- `progress.md`
  - 実施履歴の要約ログ
- `progress_weekly/`
  - 週ごとの詳細アーカイブ
- `スレッド引き継ぎファイル.md`
  - 例外時だけ使う引き継ぎメモ

## 運用ルール

- 通常時はここから読み始めない
- まずは上位の `../README.md`、`../NEXT_TASK.md`、`../開発ロードマップ.md` を見る
- 経緯確認、過去の判断理由、例外運用の確認が必要なときだけ開く
