## フォルダの見方

- `計画/`
  - 現行の中期判断。フェーズ、AI事後評価、価格精度改善の正本
  - 入口は `計画/README.md`
- `運用/`
  - 日常コマンド、通知表示、採点調整などの実務メモ
- `履歴/`
  - 進行ログ、週次アーカイブ、例外時の引き継ぎメモ
- `reports/`
  - `daily-sync` や週次の出力
  - 入口は `reports/README.md`
- `参考資料/`
  - すぐ読まない補助資料や旧版メモ
  - 入口は `参考資料/README.md`
